# Projects

This folder is for active execution work with a clear outcome.

The public starter keeps one generic example:

- [[Sample-Project-Launch|Sample Project Launch]]

## Project Rule

Create a project note when:

- the work has an owner
- the work has a deliverable
- the work needs multiple sessions
- the work should leave behind reusable lessons

When the work is finished:

1. close the project
2. move durable lessons into areas or resources
3. keep the project note only as historical context if it still helps
