# Areas

The public starter ships with three safe, generic areas:

- [[Work/README|Work]]
- [[Learning/README|Learning]]
- [[Operations/README|Operations]]

## Why These Three

- `Work` holds active outputs, responsibilities, and delivery pressure.
- `Learning` holds source ingestion, synthesis, and topic development.
- `Operations` holds maintenance, reviews, cleanup, and system health.

These are starter buckets, not rigid doctrine.

If your private instance needs different categories, reshape them there.

## Area Rule

Use an area when the topic is ongoing and requires repeated attention.

If something is temporary, keep it in:

- `00_Inbox/` if it is not processed yet
- `10_Projects/` if it is active execution work
- `30_Resources/` if it is reusable reference material
