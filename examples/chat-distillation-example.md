# Chat Distillation Example

## Original Situation

A long AI conversation started as a messy planning session for a writing project.

## Reusable Method

When a planning chat becomes useful, preserve the method, not the transcript.

## Distilled Output

- define the project in one sentence
- identify the real deliverable
- separate ongoing area notes from project notes
- create a closure trigger before execution begins

## Prompt Worth Saving

"Turn this chat into one project note, one area note, and one reusable method note."

## What Was Not Worth Keeping

- repetitive brainstorming
- low-confidence naming ideas
- transient scheduling discussion
