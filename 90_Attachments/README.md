# Attachments

`90_Attachments/` is not the default home for every raw document.

Use it for:

- `SOURCE-INDEX.md`
- small local attachments that truly need to live inside the repo
- rare exception copies

Keep most raw material outside the repo and register it in `SOURCE-INDEX.md`.
