# Agent Starter Prompts

Use these prompts when you want an agent to work from the wiki first.

## Prompt Pack

### 1. Work Triage

```text
Based on 20_Areas/Work and 10_Projects, identify the highest-leverage next step and rewrite it as a shippable action.
```

### 2. Learning Synthesis

```text
Based on my recent source notes and 20_Areas/Learning, extract the three most durable takeaways and suggest where they should be linked.
```

### 3. Operations Review

```text
Based on 20_Areas/Operations, DASHBOARD.md, and 99_System/Guides/WORKFLOW.md, identify the main maintenance bottleneck and propose a cleanup pass.
```

### 4. Inbox Processing

```text
Classify recent items in 00_Inbox into project, area, resource, or delete. Explain the reasoning briefly for each move.
```

### 5. Privacy Audit

```text
Audit this repository for public-template violations. Flag any identity, finance, immigration, contact-network, or credential-like content that should move to a private instance.
```

### 6. Starter Instantiation

```text
Treat this repo as a framework layer. Help me instantiate a private working copy while keeping the public template generic and safe.
```
