# Quick Reference

## First Run

1. Read `README.md`.
2. Read `PRIVACY.md`.
3. Read `99_System/Agent-Kernel.md`.
4. Open `DASHBOARD.md`.
5. Start from `00_Inbox/`, `10_Projects/`, or one of the starter areas.

## Area Defaults

- `Work` for output and delivery
- `Learning` for ingestion and synthesis
- `Operations` for maintenance and system health

## Maintenance Loop

- capture in `00_Inbox/`
- register raw-source folders in `90_Attachments/SOURCE-INDEX.md`
- create source notes
- update area or synthesis notes
- log meaningful changes
- lint periodically

## Public Template Safety

- keep sample content generic
- keep secrets out of version control
- do not commit personal scores, contacts, or private registries
- keep control-layer files aligned across tools

## Optional Dashboard

```bash
cd dashboard
npm install
npm run dev
```

Requirements:

- Node `>=22 <23`
- npm `>=10 <11`
