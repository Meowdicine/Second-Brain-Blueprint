# Organize Notes Prompt

```text
Help me organize the notes in my inbox.

Please:
1. Group related notes by theme.
2. Suggest whether each note belongs in Projects, Areas, Resources, or Archive.
3. Propose clearer file names when helpful.
4. Preserve nuance instead of flattening everything into generic summaries.
5. Highlight any note that should become a reusable method, prompt, or template.
```
