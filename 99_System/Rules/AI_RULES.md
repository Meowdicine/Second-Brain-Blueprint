# AI Rules

This repository is controlled through three aligned entry systems:

- `AGENTS.md`
- `CLAUDE.md`
- `.cursor/rules/*.mdc`

The shared policy source is `99_System/Agent-Kernel.md`.

## Core Rules

- Treat the repo as a public-safe starter.
- Keep private instance data out of the template.
- Preserve the split between raw sources and wiki notes.
- Prefer compounding maintenance over one-off answers.
- Keep technical filenames and interface names in English.
